# LiaScript-Integration in OPAL [Schule]

> **Praxisvortrag — OPAL User Day, Juni 2026, Dresden**
> *Wie Hochschulentwicklungen die Bildungspraxis bereichern*

Vortrag von **Prof. Dr. Sebastian Zug** (TU Bergakademie Freiberg) und
**Martin Lommatzsch** (Geschwister-Scholl-Gymnasium Freiberg) über
[LiaScript](https://liascript.github.io/) und seine Einbettung in OPAL — von der
Motivation über den Einsatz im Schulalltag bis zu den Wegen, einen LiaScript-Kurs
in OPAL bereitzustellen. Die Präsentation ist selbst ein LiaScript-Kurs.
